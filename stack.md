# Pop
*python*
```python
def print_first_word(words):
    """Prints the first word after popping it off."""
    word = words.pop(0)
    print word
```

# Pop last
*python*
```python
def print_last_word(words):
    """Prints the last word after popping it off."""
    word = words.pop(-1)
    print word
```
